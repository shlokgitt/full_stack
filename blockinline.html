This is a block level elements

Block Elements start on new line and take up the full width available.

Examples include <div>, <h1> to <h6>, <p>, <ul>, <ol>, and <li>.

This is an Inline Elements hello,Everyone

Inline Elements do not start on a new line and only takes up as much width as neccesary.

Examples include <span>, <a>, <img>, <strong>, and <em>.
